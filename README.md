# Landing Page Template (HTML CSS Website)

## Descripton

This project is a simple three-page landing page template designed as an example business/product website.

The template includes a homepage with a hero image, descriptive text, and a sign-up form; a features page with a sample pricing table and a second form; and an about page explaining the purpose of the template.

The site is built using HTML and CSS and demonstrates layout structure, forms, tables, and styling.

---

# Technical Specifications

## HTML Requirements

- Three separate HTML pages (index, features, about)
- Consistent grid/layout acros pages using Flexbox
- Uses 10+ HTML tags (header, nav, footer, h1, h2, p, img, form, input, select, button, table, tr, td, etc.)
- Includes one table (pricing table on features page)
- Includes two forms (email sign-up form on the home page and features page)
- Includes at least one dropdown menu (select inputs in both forms)
- Includes text, images, and a GIF

## CSS Requirements

- External CSS file (styles.css)
- Internal CSS (located in feature_page.html)
- Inline CSS (applied to specific elements)
- Includes at least five CSS selectors: element, class, ID, pseudo-class, and decendant
- Uses Flexbox for layout
- Includes at least two CSS animations (hero buttons hover, nav bar hover)
- Color palette uses complentary tones
